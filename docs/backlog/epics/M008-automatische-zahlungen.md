# Epic M008: Automatische Zahlungen (Taschengeld)

**Status:** ⬜ Offen (0/13 SP)

## Beschreibung

Eltern können wiederkehrende Taschengeld-Zahlungen auf der Mobile-App verwalten: erstellen, bearbeiten, pausieren und löschen.

## Business Value

Komfortable Verwaltung des Taschengelds von unterwegs. Eltern haben volle Kontrolle über automatische Zahlungen direkt auf dem Smartphone.

## Akzeptanzkriterien (Epic-Level)

- [ ] Eltern können wiederkehrende Zahlungen erstellen
- [ ] Übersicht zeigt alle aktiven und pausierten Zahlungen
- [ ] Zahlungen können bearbeitet werden (Betrag, Intervall, Tag)
- [ ] Zahlungen können pausiert und fortgesetzt werden
- [ ] Zahlungen können gelöscht werden (mit Bestätigung)
- [ ] Nächstes Ausführungsdatum wird angezeigt
- [ ] Historie zeigt alle ausgeführten Zahlungen
- [ ] Optionale Benachrichtigung vor Ausführung
