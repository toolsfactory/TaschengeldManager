# Epic M007: Geldanfragen (Kind → Eltern)

**Status:** ⬜ Offen (0/16 SP)

## Beschreibung

Kinder können Geldanfragen an ihre Eltern stellen. Eltern können Anfragen genehmigen oder ablehnen.

## Business Value

Fördert Kommunikation zwischen Eltern und Kindern über Geld. Lehrt Kinder, Wünsche zu artikulieren und Geduld zu üben.

## Abhängigkeiten

- M011 (Push-Benachrichtigungen für Echtzeit-Updates)

## Akzeptanzkriterien (Epic-Level)

- [ ] Kinder können Anfragen mit Betrag und Begründung erstellen
- [ ] Kinder sehen ihre eigenen Anfragen mit Status
- [ ] Offene Anfragen können zurückgezogen werden
- [ ] Anfragen können vor Genehmigung bearbeitet werden
- [ ] Eltern sehen alle offenen Anfragen ihrer Kinder
- [ ] Eltern können Anfragen genehmigen (mit optionaler Notiz)
- [ ] Eltern können Anfragen ablehnen (mit Begründung)
- [ ] Bei Genehmigung wird automatisch eingezahlt
- [ ] Badge zeigt Anzahl offener Anfragen
- [ ] Schnell-Anfragen mit vordefinierten Beträgen möglich
